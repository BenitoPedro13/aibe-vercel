CÓDIGO CORRETO PARA O ACCORDION ITEM:

Substitua TODO o componente AccordionItem (linhas 59-89) por:

function AccordionItem({ 
  question, 
  answer, 
  isOpen, 
  onClick 
}: { 
  question: string; 
  answer: string; 
  isOpen: boolean; 
  onClick: () => void;
}) {
  return (
    <div className="border-b border-gray-200">
      <button
        className="w-full py-6 text-left flex items-center gap-4 focus:outline-none"
        onClick={onClick}
      >
        <RiAddLine
          className={`w-6 h-6 text-gray-600 transition-transform duration-200 ${
            isOpen ? 'rotate-45' : ''
          }`}
        />
        <span className="text-lg font-bold text-black">{question}</span>
      </button>
      {isOpen && (
        <div className="pb-6">
          <p className="text-gray-600 leading-relaxed">{answer}</p>
        </div>
      )}
    </div>
  );
}

A MUDANÇA PRINCIPAL É:
- O <RiAddLine> (ícone) vem ANTES do <span> (texto)
- Assim o ícone fica no lado esquerdo da pergunta
